Bắt đầu đúng thứ tự (Local -> GitHub)

  1. Mở terminal tại folder dự án local.
  2. Chạy git init -> git add . -> git commit -m "chore: initial commit".
  3. Trên GitHub tạo repo mới (không tạo sẵn README/.gitignore).
  4. Chạy git branch -M main -> git remote add origin <repo-url>.
  5. Chạy git push -u origin main và kiểm tra code đã lên repo.
