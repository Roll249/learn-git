# Command Catalog Strategy

Mỗi lệnh trong trang `commands` cần có 4 phần cố định:

1. Cú pháp chuẩn.
2. Khi nào dùng (use-case).
3. Ví dụ thực tế.
4. Lỗi thường gặp và cách tránh.

## Nhóm lệnh bắt buộc

- Cơ bản: `status`, `add`, `commit`, `log`.
- Nhánh: `switch`, `merge`, `branch`.
- Remote: `remote`, `push`, `pull --rebase`, `fetch`.
- Nâng cao: `rebase -i`, `cherry-pick`, `revert`, `reflog`.
