# Site Map

- `/` - Trang chủ + tổng quan lộ trình + tiến độ.
- `/lessons/` - Danh sách bài học + checkpoint quiz.
- `/commands/` - Command handbook + tìm kiếm theo từ khóa.
- `/labs/` - Lab mô phỏng + tự chấm.

## Luồng chính của học viên

Home -> Lessons -> Commands (khi cần tra cứu) -> Labs -> quay lại Lessons để hoàn tất tiến độ.
