# Learning Outcomes

Sau khi hoàn thành khóa, học viên có thể:

1. Giải thích được mô hình Working Tree - Staging - Repository.
2. Tạo commit nhỏ, message rõ ràng theo chuẩn nhóm.
3. Làm việc theo GitHub Flow: branch feature -> PR -> review -> merge.
4. Xử lý conflict cơ bản khi merge/rebase.
5. Dùng `rebase`, `cherry-pick`, `revert`, `reflog` trong bối cảnh phù hợp.
6. Tự khắc phục sai sót phổ biến mà không làm hỏng lịch sử chung của team.

## Chuẩn đầu ra tối thiểu

- Hoàn thành 8/8 lesson.
- Pass tối thiểu 75% checkpoint quiz.
- Hoàn thành 4 lab mô phỏng với rubric đạt >= 3/4 tiêu chí.
