# Curriculum Blueprint

## Foundation

- B0: Từ folder local -> tạo repo mới trên GitHub -> push lần đầu.
- B1: Mental model của Git.
- B2: Repo và commit đầu tiên.
- B3: Branching và merge cơ bản.

## Collaboration

- B4: Kết nối remote và đồng bộ với GitHub.
- B5: GitHub Flow end-to-end.
- B6: Conflict resolution theo quy trình.

## Professional Workflow

- B7: Lịch sử sạch bằng interactive rebase/cherry-pick.

## Troubleshooting

- B8: Reset, revert, reflog để rollback an toàn.

## Thiết kế phiên học chuẩn

- 20% lý thuyết ngắn.
- 50% demo + thực hành có hướng dẫn.
- 30% lab theo tình huống thực tế.
