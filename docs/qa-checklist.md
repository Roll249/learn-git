# QA Checklist

## Nội dung

- [ ] Thuật ngữ Git nhất quán trên mọi trang.
- [ ] Mỗi lesson có mục tiêu đo được.
- [ ] Mỗi lesson có quiz checkpoint.
- [ ] Mỗi lab có context/task/expected/explain.

## Kỹ thuật

- [ ] Điều hướng giữa các trang hoạt động.
- [ ] Tìm kiếm command trả kết quả đúng.
- [ ] Tiến độ localStorage hoạt động và reset được.
- [ ] Không có lỗi console nghiêm trọng khi sử dụng luồng chính.

## Trải nghiệm

- [ ] Người mới có thể hoàn thành lesson đầu trong <= 20 phút.
- [ ] Văn phong giải thích ngắn, rõ, không quá tải thuật ngữ.
- [ ] Các nút hành động chính dễ nhìn trên desktop/mobile.
